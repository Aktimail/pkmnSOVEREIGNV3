Titre provisoire - Pokemon SOVEREIGN

FanGame Pokemon écrit en Python fabriqué avec Pygame et Tiled

L'objectif du projet est de recréer Pokemon le plus fidèlement possible, basé sur la cinquième génération.
Visuellement, mes moteurs de jeux ne permettent qu'une approche full 2D, ce qui n'est plus le cas au-delà de la troisième génération, qui sera donc ma référence visuelle.

le scénario n'est pas encore bien clair mais devrait exister dans un contexte de révolution, inspiré par la royauté française.
